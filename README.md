**O Que deve ser utilizado**

- Variáveis
- Operadores

- Funções
- Classes e Objetos

## Objetivo

Crie uma classe generica que represente um herói de uma aventura e que possua as seguintes propriedades:

<ul>
  <li><strong>nome</strong></li>
  <li><strong>idade</strong></li>
  <li>tipo (ex: <strong>guerreiro, mago, monge, ninja</strong>)</li>
</ul>

além disso, deve ter um método chamado atacar que deve atender os seguientes requisitos:

<ul>
  <li>exibir a mensagem: "o {tipo} atacou usando {ataque}"</li>
  <li>aonde o {tipo} deve ser concatenando o tipo que está na propriedade da classe</li>
  <li>e no {ataque} deve seguir uma descrição diferente conforme o tipo, seguindo a tabela abaixo:</li>
</ul>

se mago -> no ataque exibir (usou magia)</br>
se guerreiro -> no ataque exibir (usou espada)</br>
se monge -> no ataque exibir (usou artes marciais)</br>
se ninja -> no ataque exibir (usou shuriken)

## Saída

Ao final deve se exibir uma mensagem:

- "o {tipo} atacou usando {ataque}"
  ex: mago atacou usando magia
  guerreiro atacou usando espada
